About

Built for the
Command Line
generation.

Blackfang Labs builds privacy-first, local-only security tools for analysts, researchers, and penetration testers. Every tool works offline, ships with zero telemetry, and requires no subscription or API key to be fully functional.

The philosophy is simple: security researchers shouldn't have to send their targets to third-party APIs just to gather intelligence. Everything runs on your machine, under your control.

🔒

Local by Default

All analysis runs locally. No data leaves your machine unless you explicitly make outbound requests (WHOIS, rDNS, username checks).

⚡

No Gatekeeping

All tools are free and open. No freemium tiers, no rate-limited APIs, no account creation required.

🛠

Termux-Compatible

Every tool is designed to run on Android via Termux — full-featured security tools in your pocket.

🎯

Graceful Degradation

Missing an optional database? The tool tells you what's missing and keeps working with what it has.
